Sobre o projeto

Este projeto aborda o desenvolvimento de um jogo de tabuleiro multijogador para a disciplina de Programacao Web

Objetivos

Desenvlvoer uma pagina web responsiva e visualmente atrativa para o jogo Dama, cumprindo todos os requisitos de desenvolvimento da disciplina

Disciplina

Este projeto faz parte da disciplina INE-5646/Programacao para Web.

Recursos usados no projeto

  • Linguagens/tecnologias: HTML, CSS, JavaScript.
  • Hosting em VPS da UFSC.
